# Movie Library (Личная кинотека)

**Автор:** Дмитрий Алтухов

## Описание программы
*"Movie Library — это приложение для хранения информации о фильмах (название, жанр, год выпуска, рейтинг). Позволяет
добавлять новые фильмы, удалять старые, фильтровать по жанру и году выпуска."*

### Пример работы:
1. Ввести название, жанр, год и рейтинг фильма, после чего нажать на кнопку "Добавить фильм"
2. Проделать это несколько раз
3. После того как у вас есть несколько фильмов можно опробовать функцию фильтрации, написав нужные значения в ячейки с подписями.
4. Чтобы удалить фильм нужно нажать на выбранный вами фильм мышкой и после этого нажать на кнопку "Удалить выбранный фильм"
